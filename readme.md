# OpenGL Java Project

## Description
Ce projet est sancé être un space invider en Java utilisant OpenGL pour le rendu 3D

## Prérequis
- Java JDK (version 8 ou supérieure)
- Maven
- Make
- OpenGL
- GLFW

## Installation

1. Clonez le repository :
```bash
git clone https://github.com/MargYre/space-invaders-3D-opengl-2.git
cd opengl-test
```

2. Vérifiez que Maven est bien installé :
```bash
mvn --version
```

## Utilisation

Le projet utilise un Makefile pour simplifier les commandes de build. Les commandes disponibles sont :

- Pour tout compiler et exécuter :
```bash
make all
```

- Pour nettoyer le projet :
```bash
make clean
```

- Pour compiler uniquement :
```bash
make build
```

- Pour exécuter l'application :
```bash
make run
```
